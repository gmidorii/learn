# Perfect Java 7章 Interface

----
# 見方
- オブジェクトがどんな振る舞いをするか
- どんな機能を提供するか
- 異なるクラスが同一のインターフェースを提供する意味

----

# 境界と抽象化

- 規模が大きくなると "機能" と "役割" で分割する
- 全体を組み立てるときに詳細を意識しないようにする
	- 詳細まで見きれない
    - *部品として抽象化することが大事*

→ プログラミングの原理原則

----

# Interface
- Interfaceは型の一つ
  - Javaでは、変数/オブジェクトに型を使える
- 参照型変数の型に使う
  - 参照先のオブジェクトに対して呼べるメソッドを確定する
  - 型→メソッドの流れ
