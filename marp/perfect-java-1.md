# 静的型付け
- プログラミング言語=静的型付け
  - 動的型付けはメンテできません
  - 読めないので
- 動的との違い
	- 静的: 型の整合性を「コンパイル」時にチェック
	- 動的: 型の整合性を「実行」時にチェック
